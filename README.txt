Учебный проект.
Изучение технологии Redux.
Практическая работа c HOC, action creater, store, reducer, dispatch и остальным. 
Проброска данных из стейт, между компонентами находящимися на разной вложенности.

Как работает. Минимальные стили весь акцент на React+Redux.
SPA отражающая работу ресторанного меню.
UI: Страница с списком блюд в виде карточек. Функционал: добавления позиций в корзину,
удаление товаров из корзины, объединение одинаковых позиций в корзине, а так же суммарная стоимость набранных товаров,
Запрос данных с сервера с помощью Fetch для построения списка товаров.

Back представляет из-себя бутафорию на Json-servere,
который принимает JSON файл db.json там структура массива, а
в качестве элементов объекты c параметрами блюда.
